# LinkedIn case study assets

## Suggested headline

I built an automated Renewables Market Intelligence Radar inspired by the work of
strategy teams in the energy transition.

## Core message

The prototype converts public regulatory, competitor, supplier and electricity-market
updates into prioritised strategic signals. It uses Python for data processing,
transparent relevance scoring, unsupervised NLP clustering, a Streamlit dashboard
and an automatically generated executive brief.

## Important disclaimer

This is an independent portfolio project based solely on public information. It is
not affiliated with Vattenfall and does not represent its internal systems,
capabilities or views.

## Demo sequence

1. Show the raw public-update dataset.
2. Run `python run_pipeline.py`.
3. Open the executive HTML brief.
4. Launch the Streamlit dashboard.
5. Filter to Germany + Regulation & Policy + High urgency.
6. Open one signal and explain the implication and suggested action.
7. Show the source link and transparent score components.
8. End with the production roadmap.
